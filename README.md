# Basics
## Install virtualbox
https://www.virtualbox.org/

## Install cygwin
https://www.cygwin.com/

## Install Vagrant
https://www.vagrantup.com/

### Create VagrantFile
- Copy the VagrantFile
- Put the file in a directory
- Open up cygwin and fire `vagrant up`

# Known issues
- `vagrant up` throws provider error. Add `--provider` flag
